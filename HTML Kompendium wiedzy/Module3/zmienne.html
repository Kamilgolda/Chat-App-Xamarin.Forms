Zmienne:

Zmienne są używane do przechowywania danych. Są cztery sposoby deklarowania zmiennej:

  1. Nadaj mu nazwę i wartość.

    greeting = "Hello";

  2. Deklaracja bez nadawania wartości za pomocą słowa kluczowego let lub var. Dopóki wartosc zmiennej nie jest brana pod uwagę, JavaScript zwróci wartość jako niezdefiniowany.

    let mystery;

  3. Połącz powyższe dwa podejścia (jest to zalecany styl).

    let code = "Spang";

  4. Zadeklaruj odwołanie tylko do odczytu do wartości, używając słowa kluczowego const:

    const constant = "foo";

1. Nazwy zmiennych muszą zaczynać się od litery lub znaku podkreślenia.

2. W nazwach zmiennych rozróżniana jest wielkość liter.

Typy danych:

JavaScript rozpoznaje trzy proste typy:

  1. String

  2. Liczba

  3. Boolean

Możesz także zadeklarować zmienną i ustawić ją na null

let variableWithNullvalue = null;

Możesz określić bieżący typ danych w zmiennej za pomocą operatora typeof:

Operatory:

+ (addition)

- (subtraction)

* (multiplication)

/ (division)

% (modulus)

++ (increment)

-- (decrement)

Operatorzy przypisania:

x = y

x += y (x = x + y)

x -= y (x = x - y)

x *= y (x = x * y)

x /= y (x = x / y)

x %= y (x= x % y)

Operatory porównania

== (is equal to)

!= (is not equal to)

> (is greater than)

< (is less than)

>= (is greater than or equal to)

<= (is less than or equal to)

=== (is equal in value and in type)

!== (is not equal in value or in type)

Operatory boolowskie

x && y returns true if x and y are both true, false otherwise.

x || y returns true if either x or y or both are true, false otherwise.

!x returns true if x is false, false otherwise.

Trójskładnikowy operator warunkowy

x = (warunek)? wartość1: wartość2;

Operator ciągu

"Bo" + "om" returns "Boom".
