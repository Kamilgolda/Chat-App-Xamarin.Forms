document.querySelector(cssSelectorString) zwraca pojedynczy element, którego wartości atrybutów pasują do selektora css określonego przez cssSelectorString.

document.querySelectorAll(cssSelectoreString) zwraca kolekcję elementów, których wartości atrybutów pasują do selektora css określonego przez cssSelectorString.

Dodawanie, usuwanie i manipulowanie obiektami w DOM

  1. document.createElement(tagname)
  2. document.createTextNode(string)
  3. document.createAttribute(name, value)
  4. document.createDocumentFragment

Najprostszym sposobem jest wywołanie document.getElementbyId() w celu pobrania elementu nadrzędnego, do którego chcesz dodać obiekt, a następnie wywołać jedną z następujących metod w tym elemencie.

  1. appendChild(newNode) // dodaje nowy węzeł jako ostatnie dziecko wybranego elementu.
  2. insertBefore(newNode, existNode) // dodaje nowy węzeł do DOM przed już istniejącym
  3. replaceChild(newNode, existNode) // zastępuje istniejący węzeł potomny nowym węzłem
  4. replaceData(offset, length, string) // zastępuje tekst w węźle tekstowym. offset określa, od którego znaku należy zacząć, length liczbę znaków do zastąpienia oraz string tekst do wstawienia.

Do poruszania się po DOM można użyć następujących właściwości:

  1. childNodes, który zwraca wszystkie węzły potomne węzła.
  2. firstChild, które zwraca pierwsze dziecko węzła.
  3. lastChild, która zwraca ostatnie dziecko węzła.
  4. nextSibling, który zwraca węzeł bezpośrednio po bieżącym.
  5. parentNode, który zwraca węzeł nadrzędny węzła.
  6. previousSibling, który zwraca węzeł bezpośrednio przed bieżącym węzłem.

DOM definiuje również metody usuwania węzłów z drzewa dokumentów, w tym:

  1. removeChild(node), który usuwa węzeł docelowy.
  2. removeAttribute(attributeName) który usuwa nazwany atrybut z węzła elementu.
  3. removeAttributeNode(node) który usuwa podany węzeł atrybutu z elementu.
