Zewnętrzne style: <link rel="stylesheet" href="style.css" />

rel wskazuje typ dokumentu, do którego odwołuje się element link

media wskazuje typ urządzenia docelowego dla arkusza stylów;

Komentarze w CSS: /* */

h1 { font-size: 42px; color: pink; font-family: 'Segoe UI'; }

em { background-color: yellow; /* Yellow is a good highlight color */ font-style: italic; }

.red { color: red; }

#alert { background-color: red; color: white; }

h2.blue { color: blue; }

h2#toc { font-weight: bold; /* pogrubiona czcionka */ }

h2.blue - Zwraca wszystkie elementy h2 klasy „blue”

h2#blue - Zwraca wszystkie elementy h2 o identyfikatorze „blue”

section, h2 - Zwraca dowolny element h2 i dowolną sekcje

section h2 - Zwraca dowolne elementy h2 zagnieżdżone w elemencie section na dowolnym poziomie.

section > h2 - Zwraca wszystkie elementy h2 zagnieżdżone bezpośrednio w elemencie section

section + h2 - Zwraca wszystkie elementy h2 bezpośrednio następujące po tym samym rodzicu i dzielące się nim elementem section

section ~ h2 - Zwraca wszystkie elementy h2 następujące i współużytkujące ten sam element nadrzędny co element section

aside * { opacity : 0.6; } - zwraca zestaw wszystkich elementów w elemencie aside i i zmienia ich przezroczystosc

Selekcja atrybutów

input[type] - Zwraca wszystkie elementy input, które używają atrybutu type, niezależnie od jego wartości.

input[type="text"] - Zwraca wszystkie elementy input, w których wartość atrybutu type jest dokładnierówny ciągowi „text”.

input[foo~="red"] - Zwraca wszystkie elementy input, w których atrybut foo (na przykład atrybut klasy) zawiera rozdzieloną spacjami listę wartości, z których jedną jest dokładnie równe „red”.

input[type^="sub"] - Zwraca wszystkie elementy input, od których zaczyna się wartość atrybutu type dokładnie ciągiem „sub”.

input[type$="mit"] - Zwraca wszystkie elementy input, w których kończy się wartość atrybutu type dokładnie ciągiem „mit”.

input[type*="ubmi"] - Zwraca wszystkie elementy input, w których zawiera wartość atrybutu type podciąg „ubmi”.

input[foo|="en"] - Zwraca wszystkie elementy input, dla których wartość atrybutu foo jest równa albodokładnie „en” lub zaczyna się dokładnie od „en-”, tj. wartość plus myślnik.

Możesz łączyć selektory atrybutów : input[type="checkbox"][checked] { margin-left:50px; }

Zasady kaskadowe

-właściwość jest zawsze stosowana do zestawu elementów, dodając !important

h4 { font-weight: bold !important; }
