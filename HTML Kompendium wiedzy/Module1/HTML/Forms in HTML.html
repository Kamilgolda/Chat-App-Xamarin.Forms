<form>zawartosc </form> - wstawianie formularza

atrybuty:

action="identyfikuje adres URL strony, na którą przesłane bedą dane formularza""

method="określa sposób przesyłania danych na serwer. Prawidłowe wartości to:

• GET dla HTTP GET. Jest to wartość domyślna, ale nie jest bezpieczna

• POST dla HTTP POST. To jest preferowana wartość


accept-charset="identyfikuje kodowanie znaków danych przesłanych w formularzu"

enctype="identyfikuje typ MIME używany podczas kodowania danych formularza gdy metodą jest POST"

target="określa, gdzie przeglądarka wyświetli stronę akcji; prawidłowe wartości to _blank, _parent, _self i _top"


kontrolki formularza:

<input /> - wstawianie kontrolki

type="text" -Jednowierszowe pole tekstowe

type="password" -Jednowierszowe pole tekstowe, w którym tekst wprowadzony do pola jest zastępowany gwiazdkami

type="hidden" -Pole niewidoczne dla użytkownika

type="checkbox" checked -Pole wyboru Zapewnia wybór tak / nie :checked -domyślnie zaznaczone.

wybor1wybor2
type="radio" -Pole wyboru radiobutton, dopisując name grupujemy pola.

type="reset" -Przycisk resetowania. Kliknięcie tego przycisku resetuje wszystkie pola do ich wartości początkowych.

type="submit" -Przycisk przesyłania. Kliknięcie tego spowoduje przesłanie bieżących wartości formularza

type="image" -Obraz do użycia jako przycisk przesyłania. Użyj atrybutu src, aby zidentyfikować obraz

type="button" -Przycisk. Nie ma to domyślnego zachowania i może być użyte do uruchomienia skryptu, kiedy na przykład kliknięty.

type="file" -Kontrola pliku. Zapewnia sposób przesłania pliku do serwera

<textarea> -generuje dowolne, wielowierszowe pole edycji tekstu jawnego; użyj rows i cols, aby ustawić jego rozmiar.

<select> -definiuje pole listy lub listę rozwijaną. ; multiple - użytkownik może wybrać więcej niż jeden element ; <option>> - element ; selected -domyslnie wybrany ; value - wartosc ktora ma zostac wyslana

<button> -definiuje przycisk. Użyj atrybutu type, aby wskazać, czy jest to przesyłanie, resetowanie czy przycisk (nic nie robi). Domyślnie jest to przesyłane.

Elementy układu formularza

Możesz zastosować znaczniki <p> ​​i <div>, aby zastosować podstawowy układ do formularza.

Personalia:
<fieldset> -identyfikuje grupę formantów w formularzu. Przeglądarka odzwierciedla to, rysując ramkę wokół zawartości, oznaczając ramkę nazwą. Ta nazwa jest ustawiana za pomocą elementu <legend>

<label> -identyfikuje etykietę tekstową powiązaną z kontrolką formularza, Czyni to albo przez obejmowanie zarówno tekstu, jak i kontrolki, lub otaczając tekst i ustawiając atrybut for na id kontrolki
