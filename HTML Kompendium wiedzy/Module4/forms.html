Element form ma następujące atrybuty

  • name - atrybut jest używany przez serwer do odwoływania się do pól w formularzu podczas przetwarzania.
  • action - gdy użytkownik prześle formularz na serwer. To jest adres URL, na który formularz zostanie wysłany. Jeśli pominiesz adres URL, formularz zostanie wysłany na bieżący adres URL strony internetowej.
  • method - Metoda stosowana do wysyłania danych. Podczas przesyłania danych na serwer za pomocą formularza, atrybut powinien normalnie być ustawiony na post.

fieldset - Ten element służy do rysowania pola wokół zestawu pól i dodaje etykietę do zestawu za pomocą elementu <legend>. fieldset jest opcjonalny, ale może być bardzo przydatny w większych formularzach, w których użytkownik może zobaczyć etykietę sekcji.

dobrą praktyką jest dodawanie elementu <div> dla każdego pola w formularzu

Do diva dodajesz element <input> w celu gromadzenia danych. Element wejściowy powinien zawierać następujące atrybuty:

  • id and name attribute.
  • type attribute
  • placeholder attribute

Większość elementów wejściowych ma powiązany element label

Na końcu element input z typem danych „submit”. Ten znacznik tworzy klikalną kontrolkę, która wysyła dane z formularza do serwera.

Możesz użyć elementu <datalist> z wprowadzaniem tekstu, aby zapewnić opcje autouzupełniania.

<textarea> formant jest renderowany jako obszar wielowierszowy przy użyciu czcionki o stałej szerokości. Atrybuty cols i rows kontrolują jego szerokość i ilosc wierszy Możesz ustawić jego atrybut maxlength, aby ustawić maksymalną liczbę znaków.

Element <select> jest pojemnikiem na pewną liczbę stałych elementów. <option> predefiniują dane wejściowe Aby pogrupować opcje w mniejsze listy logiczne, użyj element <optgroup>

Atrybuty inputa

atrybut autocomplete - Jeśli użytkownik wprowadził wcześniej te same dane, poprzednia wartość jest automatycznie kopiowana do danych wejściowych przez przeglądarkę.

Email: Password:

required wskazuje, że pole jest obowiązkowe.

pattern umożliwia zastosowanie wyrażenia regularnego do pola wprowadzania tekstu, aby się upewnić jest zgodny z określonym wzorem.

placeholder umieszcza zawartość tymczasową w polu tekstowym
