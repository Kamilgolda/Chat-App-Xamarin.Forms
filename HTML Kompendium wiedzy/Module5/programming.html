Arrow functions Nazywa się je tak, ponieważ używają znaku =>. Strzałka.

Są to funkcje są anonimowe, jednowierszowe mini funkcje, i są bardziej zwięzłe niż funkcja w starym stylu wyrażenia.

podstawowa konstrukcja funkcji strzałki to:

(parameters) => { statements(instrukcje) }
