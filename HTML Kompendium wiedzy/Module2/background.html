background-image umożliwia określenie adresu URL obrazu użyj jako tła dla wybranych elementow.
background-image:url('../images/pattern.jpg');

background-size umożliwia ustawienie wysokości i szerokości tła Użyj wartości określonych w pikselach lub jako procent wysokości i szerokości określonego element.
background-size: 40px 60px; /* 40px wide, 60px high */

background-color umożliwia ustawienie koloru tła elementu. możez określić kolor jako wartość RGB lub jako jedną ze 147 wstępnie zdefiniowanych nazw kolorów
background-color : green;
background-color : #00FF00;
background-color : rgb(0, 255, 0);

background-position umożliwia ustawienie położenia obrazu tła w elemencie. Właściwość przyjmuje dwie wartości: pierwszą dla osi x i drugą dla osi y. Zestaw oba jako wartości bezwzględne (góra, lewo, dół, prawo, środek), wartości procentowe lub piksele.
background-position : left top; /* Image locked into top left corner of element */
background-position : 100% 100%; /* Image locked into bottom right corner of element */
background-position : 8px 8px; /* Image starts 8px from left and 8px from top of element */

background-origin umożliwia ustawienie, który z modeli pudełkowych ma zawierać tło. Możliwe wartości to content-box (the default), paddingbox, and border-box.
background-origin : border-box;

background-repeat umożliwia ustawienie sposobu powtarzania obrazu tła za wybranym elementem, jeśli jest mniejszy niż wybrany element. Możliwe wartości: repeat(the default), repeat-x, repeat-y and no-repeat.
background-repeat : repeat-x; /* Repeat background image only horizontally */
background-repeat : no-repeat; /* Don't repeat the image */

background-attachment pozwala ustawić, czy obraz tła przewija w górę i w dół ze stroną lub pozostaje na miejscu. Możliwe wartości scroll (the default) and fixed.
background-position : fixed;

Prawidlowa kolejnosc skrótów dla background:

  1. background-color
  2. background-position
  3. background-size
  4. background-repeat
  5. background-origin
  6. background-clip
  7. background-attachment
  8. background-image

background : transparent repeat-x url('fluffycat.jpg'); == background-color : transparent; background-repeat : repeat-x; background-image : url('fluffycat.jpg');
